# Personal Portfolio Website

**Developer:** Kalyan Bathina

## Overview

This project is a personal portfolio website designed to showcase my skills, projects, and contact information. The website provides a modern and responsive interface where visitors can explore my work and learn more about me.

## Features

* Responsive design for desktop and mobile devices
* Navigation menu for easy section access
* About section with personal information
* Projects section displaying my work
* Skills section highlighting technical abilities
* Contact section for communication
* Dark/Light theme toggle

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap

## Project Structure

```
portfolio-project
│
├── index.html
├── style.css
├── script.js
└── images
```

## Installation / Usage

1. Download or clone the repository
2. Open the project folder
3. Run the project by opening **index.html** in a web browser
